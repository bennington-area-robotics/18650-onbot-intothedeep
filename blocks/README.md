# Blocks Programs

FTC Blocks programs exported from the Control Hub.

## Active Programs

| File | Description |
|------|-------------|
| `ArmPowerTest.blk` | Arm power testing |
| `armTest.blk` | Arm functionality test |
| `encoderTest.blk` | Encoder testing |
| `extensionTest.blk` | Extension mechanism test |
| `MotorTest.blk` | Motor testing |
| `moveWrist.blk` | Wrist movement control |
| `RunLFront.blk` | Left front motor run |
| `SpoolTest.blk` | Spool mechanism test |
| `touchSensorTest.blk` | Touch sensor testing |

## Archive

The `archive/` folder contains older or experimental programs that are no longer in active use.

## Usage

To use these programs:

1. Connect to the Control Hub
2. Open the Blocks editor in a web browser
3. Import the `.blk` file

To export changes:

1. Download the program from the Blocks editor
2. Save to this directory
3. Commit and push
